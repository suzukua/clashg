# ClashG(AX86U开发测试，理论上支持其他HND平台)
依据GFW域名用dnsmasq进行分流+部分国外IP，对国内流量无任何影响(高稳定、高可靠、高容错方案)
## Features
- 基于[Clash.Meta](https://github.com/MetaCubeX/Clash.Meta)
- 支持局域网内TCP/UDP透明代理(TPROXY)
- 支持Shadowsocks入站,UDP+TCP,IPV6+IPV4
- 入站支持TFO
- 支持自定义规则地址
- 支持自动更新geoip和gfw规则
- 支持手动编辑clash配置文件
